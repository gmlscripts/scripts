string_reverse
==============

Returns a given string with the characters in reverse order.

    s = string_reverse("hello, world!");    //  "!dlrow ,olleh"
    s = string_reverse("\t\r\n");           //  "\n\r\t"
    s = string_reverse(" n\r\t\ ");         //  " \t\rn "
