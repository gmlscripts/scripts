bin_to_fraction
===============

Converts a string of binary digits to a fractional value.

    val = bin_to_fraction("00100");  //  val == 0.12500
    val = bin_to_fraction("10101");  //  val == 0.65625
    val = bin_to_fraction("11111");  //  val == 0.96875
