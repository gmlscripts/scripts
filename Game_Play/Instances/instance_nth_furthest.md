instance_nth_furthest
=====================

GameMaker can already tell you the [instance furthest] from a point, but what
about the 2nd furthest? Or the 3rd furthest? Or the nth furthest? This script
answers that question.

demo: instance_nth_furthest 512 64

script: instance_nth_furthest.gml

contributors: xot

[instance furthest]: https://manual.gamemaker.io/monthly/en/#t=GameMaker_Language%2FGML_Reference%2FAsset_Management%2FInstances%2Finstance_furthest.htm
