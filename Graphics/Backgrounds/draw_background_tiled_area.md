draw_background_tiled_area
==========================

Note: This script is obsolete. GameMaker no longer supports the legacy Background image asset type.
Please use [`draw_sprite_tiled_area()`](/script/draw_sprite_tiled_area) instead.

Draws a repeated background image, tiled to fill a given region and with
a given offset.

demo: draw_background_tiled_area 512 192
Mouse over to move origin offset.
