implode
=======

Returns a string of elements from a given array
of strings and separated by a delimiter.

    array[0] = "Juliett"
    array[1] = "Victor"
    array[2] = "Mike"
    array[3] = "Charlie"
    array[4] = "Romeo"
    array[5] = "Oscar"
    
    names = implode(",", array);
    
    //  names == "Juliett,Victor,Mike,Charlie,Romeo,Oscar"
