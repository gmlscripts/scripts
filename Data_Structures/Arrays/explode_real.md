explode_real
============

Returns an array of real values parsed from a given
string of elements separated by a delimiter.

    numbers = "10,21,13,3,18,15";

    array = explode(",", numbers);

    //  array[0] == 10
    //  array[1] == 21
    //  array[2] == 13
    //  array[3] == 3
    //  array[4] == 18
    //  array[5] == 15
