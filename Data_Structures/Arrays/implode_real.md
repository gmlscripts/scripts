implode_real
============

Returns a string of elements from a given array
of real values and separated by a delimiter.

    array[0] = 10;
    array[1] = 21;
    array[2] = 13;
    array[3] = 3;
    array[4] = 18;
    array[5] = 15;

    numbers = implode(",", array);

    //  numbers == "10,21,13,3,18,15"
